# 目标2：陈叙川 — 修订版大纲

## 目标档案

| 项目 | 内容 |
|------|------|
| 姓名 | 陈叙川 |
| 年龄 | 27岁 |
| 身份 | 宁城第一医院心外科主治医师 |
| 性格 | 温和、边界感强、职业病级别的自律。27年从未打破过习惯——直到遇到她 |
| 女友 | 温晓棠，26岁，同院儿科医师。大学同窗六年，已订婚。独立、骄傲、矜持 |
| 他未被满足的需求 | 被照顾（不是被要求）。被打破惯性（他不知道自己困在习惯里） |

## 和目标1的核心差异

| | 目标1（陆行舟） | 目标2（陈叙川） |
|------|------|------|
| 他的需求 | 被需要、被崇拜 | 被照顾、被打破惯性 |
| 他的恐惧 | 她不选他 | 习惯被打破后的失控感 |
| 她的身份 | 崇拜他的穷学妹 | 不需要他负责的发小 |
| 原配类型 | 高调晒男友的千金 | 骄傲矜持的同事 |
| 原配怎么输的 | 她只晒他不关心他 | 她太独立不给他照顾她的机会 |
| 她拿走的第一次 | 初吻、第一次失控 | 第一次打破边界、第一次对未婚妻说谎 |
| 心锚 | 咬下唇 | 赤脚踩木地板的声音 |

---

## 六步撬动法（目标2版）

```
第一步：不攻击现有关系 → "发小"身份天然无害，她只是老朋友
第二步：被看见 → 她说"你瘦了"，这是温晓棠六年没说过的话
第三步：制造对比 → 她在家里等他 vs 温晓棠永远在加班
第四步：习惯植入 + 间歇强化 → 借住后他习惯家里有人 → 她突然说要搬走
第五步：肉体破防 → 醉酒夜晚 + 电话处刑（升级版）
第六步：让他自己决定分手 → 她没说过"分手"，但他自己拨的电话
```

---

## 章节规划（7集）

### 第1集：发小（原第9章）
**六步对应**：第一步 不攻击现有关系

**关键场景**：
- 接风宴上她当众替他擦嘴角。温晓棠质疑——"你不觉得你和我男朋友太亲密了吗"
- 陈叙川的第一反应是为她解释——"她从小就这样"
- 她低头认错："对不起我真的没注意…"但眼神飞快瞥了一眼陈叙川

**新增：皮格马利翁植入**
- 饭后她拍拍他的肩说"叙川你从小就是最可靠的"——给他贴了一个他永远不能打破的标签。以后他任何拒绝她的念头都会被这句话挡住。

---

### 第2集：医院（原第10章）
**六步对应**：第二步 被看见

**关键场景**：
- 她以"心慌"为由去他医院挂号看病
- 听诊器贴上胸口时——他能听到她的心跳，但她也能听到他的呼吸变重了
- 他说"你瘦了"——她穿得很少，头发被风吹乱
- 护士台开始传播流言

**新增：间歇强化启动**
- 看完病之后她连续三天没联系他。他翻开和她的微信对话框看了好几次。第一次发现自己在等一个"病人"的消息。

---

### 第3集：借住（原第11章）
**六步对应**：第二步→第三步 制造对比

**关键场景**：
- 她以"房子装修没地方住"为由请求借住
- 他犹豫——内心边界感说不合适，但嘴上说的是"就几天"
- 她在他家穿他的T恤从浴室出来
- 温晓棠知道后质问——他的解释是"她没别的地方可去"

**新增：心锚植入**
- 她在他家里永远赤脚走路。木地板上有她赤脚的脚步声——轻轻的、吧嗒吧嗒的。她走后很久，他每次回到家还会下意识等着那个声音。但木地板永远是安静的。

**新增：推拉回合**
- 住到第三天她说"我这个周末找房子搬走，不能一直麻烦你。"他脱口而出"不急"——说出口之后自己愣了一下。

---

### 第4集：习惯（原第12章）
**六步对应**：第四步 习惯植入

**关键场景**：
- 每天早上两份早餐放在餐桌上
- 深夜他应酬回来，餐桌上有一盒外卖和一张便签——"热好了。记得吃。"
- 她发潮汐动态"等一个人"——他点了赞。温晓棠私信他"你从来不给我点赞"

**新增：富兰克林效应的推进**
- 系统列出他目前的投入——让她借住了他的卧室；每天回家吃她做的饭；对未婚妻说了第一次谎（"她只是暂住"）；凌晨回家有人等。他投入越多，越难喊停。

**新增：温晓棠楼下目击的内心描写**
- 温晓棠看到林浅浅靠在陈叙川肩上时，她转身走了。但她不是愤怒——她是害怕。她意识到自己过去六年从来没有在这个男人肩上靠过。不是他不让。是她没有。

---

### 第5集：对峙（原第13章）
**六步对应**：第四步→第五步 间歇强化 + 推拉升级

**关键场景**：
- 温晓棠约林浅浅谈话——"你不觉得不合适吗"
- 林浅浅全程示弱——"对不起""我明天就搬"
- 回头对陈叙川说——"你女朋友让我搬走。我不想让你为难。"

**新增：推拉升级**
- 她说要搬走的时候是真的在收拾行李箱。不是假的——她把牙刷和睡衣都装进去了。陈叙川站在门口看着她收拾。她没有看他。她在赌——赌他会不会开口留她。行李箱拉链拉上的那一刻——他说："你不用搬。"她把行李箱放下了。但不是马上。隔了三秒。这三秒是他这辈子最长的三秒。

---

### 第6集：电话（原第14章）
**六步对应**：第五步 肉体破防 + 第六步 让他自己分手

**关键场景**：
- 他应酬喝醉，她去接。在车里/家里，他吻了她。
- 温晓棠来电。她按住他的手——"别接。"
- 连打三个。她拿起电话放在他耳边。温晓棠正在说"叙川我今天好累"——
- 她用刚好能被听到的气声问"谁呀"
- 电话那头三秒死寂。他说："我们分手吧。"
- 电话挂断。她吻了他的嘴角——"乖。"

**新增：他的内心——"我自己选的"**
- 挂断电话后，他低头看着怀里的人。她笑了。那个笑容不是胜利——是"你终于做了"。他对着那个笑容，把涌到嘴边的"我到底在干什么"咽回肚子里。这是他自己的选择。没有人逼他。她只是问他："谁呀。"

---

### 第7集：余温（原第15章）
**六步之外**：蔡格尼克收尾 + 坏 + 离场

**保留原有的"坏"**：
- 穿温晓棠的白大褂照镜子→"好看吗"
- 把温晓棠的杯子扔进垃圾桶→"旧的该换了"
- 医院走廊和眼眶红肿的温晓棠擦肩而过→她笑了一下

**新增：系统损失清单**
- 系统弹出他不可逆的损失——未婚妻（已结束）、职业声誉（医院流言）、情感习惯（余生回家都会下意识听赤脚踩木地板的声音）
- 附加记录——"宿主从未答应过目标的求婚。目标提出的'嫁给我'在文本中无任何宿主回复记录。蔡格尼克指数：97%。"

**新增：蔡格尼克收尾**
- 他醒来发现她消失了。所有她的东西都不见了——她的牙刷、她的睡衣、她的行李箱。只剩垃圾桶里温晓棠的那个杯子。还有木地板——永远安静的木地板。
- 微信最后一条消息：「任务完成。谢谢你呀，叙川哥哥。」
- 他后来每天下班回到家，推开门的瞬间还是会下意识听——有没有赤脚踩木地板的声音。听了很久。永远是安静的。
