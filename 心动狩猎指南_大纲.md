# 《心动狩猎指南》20 集大纲

## 世界观与系统

```
系统：心动狩猎
功能：① 显示目标好感度 ② 分析目标偏好 ③ 推荐攻略人设
规则：每攻略一个 S 级目标解锁一项新技能
奖励递进：魅惑光环 → 读心碎片 → 距离感知 → 未知终局奖励
惩罚：无。系统不惩罚宿主，只奖励——因为它要的不是"任务完成"，是"越玩越大"
```

## 核心设定

```
女主时染：S 级天赋，但天赋不是手段——是她读得懂每个男人心里缺的那一块
她缺什么，就给什么。她怕什么，就避什么。

性格不是固定的——对不同类型的男人用不同的脸。
但骨子里是同一个人：不走心、不内疚、享受狩猎过程本身。
```

## 分集大纲

### 第一幕：狩猎开场（Ep 1-4）

| 集 | 钩子开场 | 手法爆发 | 爽点 | 结尾钩子 |
|---|---|---|---|---|
| **1** | 分手现场→系统激活 | 十分钟内连撩两人：江边的陆司砚（暖愈）、顾砚声（同类） | 发现"性格适配"天赋——对不同男人用不同脸 | 系统提示：目标 002 曾有三年恋情——女友自称是时染的双胞胎妹妹。但她是独生女 |
| **2** | 中午给陆司砚送饭 | 陆司砚看到围巾吃醋，好感度从 34% 跳到 41%——让他酸比让他甜管用 | 发现：嫉妒是最快的涨分手段 | 顾砚声问她"你等的人是谁"——她没回答。但他看着她的眼神在说：我知道你在演 |
| **3** | 两部手机同时收到消息——陆司砚约晚餐，顾砚声约看展 | 两边都答应，错开时间。结果晚餐和看展在同一条街，只隔一栋楼 | 极限时间管理，差三十秒撞车——她借一辆出租车从餐桌逃到展厅 | 逃出餐厅大门时撞上一个人——一个戴无框眼镜的男人拦住了她："你刚才在陆司砚那桌说的话，我全听到了。" APP 弹窗：目标 003 江辞野触发。身份需求判定中——无法分析 |
| **4** | 江辞野追着她不放，说自己是电竞职业选手，直觉比普通人准 | 对电竞男不用手段——所有手段他都能读。她换了一张脸：完全不撩他，反而怼他 | "你以为我在套路他们？你有没有想过，是他们需要一个套路——不然怎么解释自己为什么沦陷？" 江辞野被噎住了 | 江辞野说：那我也缺——我缺一个能赢我的人。APP 好感度：江辞野从 0 跳到 22%。三个男人，同一天，都在涨 |

### 第二幕：四线并行（Ep 5-8）

| 集 | 核心事件 | 手段教学 | 爽点 | 钩子 |
|---|---|---|---|---|
| **5** | 陆司砚公司年会——他发现顾砚声也在场，两人第一次正面遭遇 | 先发制人：在陆司砚开口前，她主动走到两人中间："给你们介绍一下——这位是我的羽毛球教练推荐的……" 一句废话把两个人的关系岔开 | 两个男人都觉得自己是"被正式介绍的那个" | 年会上第三个人走进来——是陆司砚的私人心理顾问温屿。他的眼睛在她身上停了四秒。APP 弹出目标 004 |
| **6** | 顾砚声画展开幕——她发现他的画里全是她 | 对艺术男：不要感动，要沉默。她站在画前沉默了三分钟，他站在她身后沉默了三分钟，最后她开口："你画错了一个地方。我没那么脆弱" | 她把他的真心话堵回去——越不领情，他越着迷 | 画展门口，江辞野粉丝举着灯牌路过，其中一个灯牌写着"辞野哥哥的命中注定"——照片是时染。不知道谁拍的 |
| **7** | 照片事件引爆——四个男人同时看到热搜 | 不澄清、不解释、不回应任何一个人的质问。让他们自己吵 | 江辞野的粉丝在网上扒她，反而帮她制造了"万人迷"的人设——陆司砚看到后开始主动表白 | 温屿给她发了一条消息："你不需要来看病。但如果你愿意来，我只是想听你说话。我是心理医生——但我看不懂你" |
| **8** | 温屿的诊所——她装病进去，结果被反杀 | 对心理医生不是撒谎——是让他以为自己看穿了她。她编了一个"童年缺爱的故事"，他拆穿了："这是你刚编的对吧" | 被拆穿的瞬间她没有慌，反而笑了："对。那你告诉我，真的那个在哪" | 温屿沉默了很久："你有病。但不是你想的那种——你是'反社会型人格的娱乐型变体'。说白了，你不是感受不到，你是不在乎" |

### 第三幕：翻车与翻盘（Ep 9-13）

| 集 | 核心事件 | 手段爆发 | 爽点 | 钩子 |
|---|---|---|---|---|
| **9** | 温屿的诊断让她愣了 48 小时——然后她想到一个问题 | 去找温屿："你是来治我的，还是来追我的？" 温屿："都是" | 反制心理医生：他以为自己在对她诊断，其实她已经反过来在用他的诊断当一个新角色——"被你看穿了，你还离得开我吗" | 温屿的诊所记录被人调取——调取人是顾砚声。他已经开始查她了 |
| **10** | 修罗场前夕——四个男人在同一天、同一栋楼（陆司砚公司、顾砚声画廊、江辞野比赛、温屿诊所——全集中在 CBD 方圆两公里） | 她设计了四个人的时间差，精确到分钟。但最后一步出了差错——江辞野比赛提前结束 | 她用江辞野的粉丝围堵转移注意力，顺手把顾砚声推进了陆司砚的电梯——两人在电梯里聊了十二层楼 | 电梯门开，陆司砚说："她骗了我们所有人。但我没有不高兴——你知道为什么吗？" 顾砚声："你说" |
| **11** | 陆司砚和顾砚声联手了——他们决定一起把她约出来 | 她赴约，一个人。他们两个人坐在对面 | 她说："你们在等我崩溃吗？那我帮你们省点时间——我不觉得我做错了什么。你们要道歉的话我可以给。但你们得先说明白——道什么歉" | 陆司砚和顾砚声对视了一眼。没有人说得出"你骗了我的感情"——因为她从头到尾都没说过喜欢他们。她只是让他们以为有可能 |
| **12** | 江辞野公开表白——在直播里说"我喜欢一个女生"。全网扒出是时染，然后连带扒出陆司砚和顾砚声 | 全网吃瓜——她成了"同时撩三个男人的女人"。她没躲，反而开了小号，在评论区里跟网友一起分析自己 | 不是洗白——是让吃瓜变成流量，流量变成她的筹码 | 温屿给她发了一条消息："别人都在骂你，但你知道吗——你反而更受欢迎了。这就是你的人设：坏到让人想看看你到底多坏" |
| **13** | 系统弹出一条她从没见过的提示：终极任务已解锁。进度：4/4 S 级目标攻略完成。请选择最终奖励——①删除系统，回归正常 ②绑定永久，成为系统管理员 | 她选了 ②。系统说："早就知道你会选这个——因为从第一集开始，我测试的不只是你能不能攻略男人。我测试的是——你会不会爱上攻略本身" | 系统的真相：它是一个"猎人选拔系统"，上一任管理员退休了，它在找继任者。时染是它筛了一万多人才找到的 | APP 界面变了——不再显示目标。显示的是：管理员模式。可查看附近所有猎物的好感度。可下发任务给其他宿主。可——修改好感度数值。她对着屏幕看了很久。然后她打开了"附近猎物"名单 |

### 第四幕：终局（Ep 14-20）

| 集 | 核心 |
|---|---|
| **14** | 成为管理员的第一天——她看到四个男人的实时好感度，没有一个人低于 60%。她可以修改这些数字。但她没有。她关了界面 |
| **15** | 江辞野的战队经理来找她——邀请她做战队公关。她答应了。不是因为需要工作，是因为下一批猎物在那里 |
| **16** | 温屿辞职了——他说他不能再当心理医生，因为他诊断了太多人，只有一个人让他觉得"诊断是没用的"。他来找她，不是为了追她——是想跟她做合伙人。一个心理医生和一个猎人，开一家"情感咨询公司" |
| **17** | 公司开张。第一批客户是三个被渣男伤害的女生。她教她们的不是"怎么挽回"，是——"怎么让他以为你在挽回，其实你在练手" |
| **18-19** | 新猎物：一个新出道的顶流爱豆，全网女友粉千万。他的经纪公司花大价钱请她——不是攻略他，是攻略一个想蹭他流量的女演员，让她知难而退。她接了这个活，反手把那位女演员招进了自己的公司 |
| **20** | 结局：她没有跟任何一个男人在一起。她在城市最高楼的顶层开了一间工作室，系统投射出来一面全息屏幕——几百个头像在上面浮动，每一个都是一段可以开始也可以终止的狩猎。温屿坐在隔壁写了本书《论反社会人格的情感应用》。顾砚声给她画了一幅画挂在墙上——画的是她们的第一次见面，她站在江边，笑得轻飘飘的，像一个没有重量的谜 |

**终场**：时染站在落地窗前，手机屏幕亮。APP 弹出一行——

`新目标检测：下一城市。难度：SSS 级。建议立即出发。`

她拿起包，对着镜子补了个口红。镜子里的人笑了一下。和第一集那个被分手时一模一样的笑——不同的是，这次是她自己选的。
