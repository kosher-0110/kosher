# 《暗恋倒计时》目标三大纲

## 目标档案

| 项目 | 内容 |
|------|------|
| 姓名 | 江时晏 |
| 年龄 | 26岁 |
| 职业 | 顶流男演员，一线小生，微博粉丝3000万+ |
| 性格 | 外表阳光暖男（人设），私下冷淡疏离，对陌生人极度警惕 |
| 伴侣 | 纪诗然，28岁，同为演员，两人因戏生情，公开恋情三年，被称为"内娱模范情侣" |
| 攻略难点 | 身边永远有经纪人/助理/粉丝围堵，私人空间极少。对主动贴上来的女性有天然警惕 |
| 系统建议 | 不能走"追求"路线。必须让他主动。需要一个让他无法拒绝的日常接触场景 |
| 时限 | 90天 |

## 女主新身份

- 姓名：林浅浅（系统覆盖）
- 身份：新人演员，刚从戏剧学院毕业，签了宁城一家中型经纪公司
- 背景：系统伪造了戏剧学院学历和两部网剧配角经历
- 外貌：系统微调——保留所有原本特色，但气质做了"镜头优化"。在摄像机面前有一种天然的故事感，导演一看就说"她长了一张该红的脸"
- 女儿：绵绵，三岁，继续跟随。对外宣称为"表姐的孩子，帮忙照顾"（隐藏单亲妈妈身份，避免影响事业）
- 经纪人：系统安排了一位四十岁的女性经纪人"陈姐"，负责帮她接通告和处理日常

## 接近方式

女主被选入江时晏正在拍摄的新剧《长夜》剧组，饰演女三号——男主的初恋白月光。角色只有五集戏份，但和男主有大量交集：对视、拥抱、以及一场吻戏。

---

## 核心手段（全新设计，避免与前两线重复）

### 手段1：真演假做——利用"表演"掩护真情
- 她演的是他的初恋白月光。剧本里她对他又爱又恨、欲言又止、最终离开。
- 她把这个角色演得太真了。真到导演说"cut"之后他看她的眼神还没切换回江时晏本人。
- 她利用的是"我在演戏"这个终极掩护——所有人看着她对他深情凝视，但没有人能说那不是表演。
- **突破节点**：某次高密度感情戏后，两人回到各自的休息室。半小时后他在走廊尽头看到她在角落里站着，眼眶是红的。他问她怎么了。她说"刚才那场戏，我在里面出不来了。"他以为她是在说角色。但不是。她说的是她演的那个人——对他做的一切，都是她想做的。只是角色替她挡了。
- **与前两线区别**：目标1是用"学"做掩护，目标2是用"朋友"做掩护——这里是"我在演戏"做掩护。比前两个都更难戳破，因为"演得好"本身是职业素养。

### 手段2：直播事故——镜头前的不受控
- 剧组安排了一场宣传直播。男女主和其他演员在直播间和粉丝互动。直播间在线300万人。
- 在玩游戏环节，主持人让女三号对男主说一句角色台词。她选了一句——"我从来没有后悔过喜欢你。"
- 她说这句台词的时候，看着他的眼睛。不是对着镜头。是对着他。直播间弹幕在那一秒炸了——"她看的是本人不是角色""这是在表白吧""你们看到江的反应没有"
- 他愣了一下才接上。那是他整场直播里唯一一次表情管理失败。
- 直播不能剪辑。300万人看到了那一瞬间。CP粉截图做成gif，转发破万。
- **与前两线区别**：前两线的舆论是延迟爆发的（论坛、医院流言），这里是实时直播+百万观众，压力是即时的、不可逆的。

### 手段3：模仿原配——反向摧毁
- 她研究了纪诗然所有的公开资料。穿着风格、说话节奏、小动作。
- 开始在片场微妙地模仿纪诗然——围巾的系法和纪诗然同款，喝咖啡时搅拌的方向和纪诗然一样，说某一个词的方式和纪诗然重合。
- 这些细节太细了，普通人看不出来。但江时晏看出来了。他每次看到都会心里咯噔一下——"她刚才那个动作，和诗然一模一样。"然后他会纠正自己："巧合吧。"
- 直到有一天他看到她随手绑头发的方式——和纪诗然在家的日常发型一样。一个她在外面从来不会做、只有他见过的方式。他愣住了。那瞬间他明白了——她不是在模仿。她是在替代。
- **与前两线区别**：不是"对比"原配——而是让自己变成原配的影子。他爱的好多细节，从另一个女人身上长出来了。这个手段前两线完全没用过。

### 手段4：私生饭/全网黑——让他来保护她
- 随着她在剧组的戏份增加，他的极端粉丝开始网暴她——"心机女配""抢戏""借江老师上位"
- 她的微博评论区被刷屏骂了几万条。她一条都没有删。
- 她发了一条微博：「谢谢大家的关注。我是一个新人，还有很多不足，会继续努力的。其他的就不回应了。」
- 这条微博底下骂的更多了。但她回复了其中一条骂她的——她说"谢谢你的鞭策，我会加油的。"
- 江时晏看到了。他被她的体面震住了。一个被骂了几万条的新人女演员，没有卖惨，没有关评论，而是回了其中一条说"谢谢鞭策"。
- 他发了一条微博：「请理性追星。对同组演员的恶意已经超过了底线。这是我最后一次回应这件事。」
- 他发的这条帮她挡了子弹。但同时也等于——他公开表态了。为了她。不是角色。是私下的她。
- **与前两线区别**：前两线都是女主让男方在自己和原配之间选。这一次是让男方在她和"他自己的粉丝"之间选。被一个人保护，和被一个人对抗全世界来保护，分量不同。

### 手段5：合约/综艺的不可拒绝性
- 剧组安排两人参加一档户外真人秀做宣传。节目组为了效果，把两人组成了一个任务组。
- 任务包括：两人共骑一辆自行车（她坐前杠）、用吸管共同喝完一杯饮料、她被蒙上眼睛由他指挥走过障碍。
- 这些环节是节目组设计的，不是她设计的。但她在每一个环节里都不设防——她完全信任他。蒙着眼睛走障碍时，她的小指紧紧勾着他的小指。那个镜头被弹幕刷了满屏的"我死了"。
- 他不能拒绝——因为这是工作。他的身体在配合节目组，但他的心跳在配合她。每一个环节结束，他看她的眼神都在变。
- 效果：利用"工作安排"的不可拒绝性，把他的身体拉进亲密距离。比前两线的"借住""请教"都更不可推脱。

### 手段6：吻戏——唯一不喊卡的人
- 两人的吻戏拍了三条。导演喊了两次卡。第一次——角度不对。第二次——灯光有问题。第三次——导演没喊卡。是摄影师先说"好了"的。因为导演在看监视器，忘了喊。
- 拍完之后她站在原地，没有马上离开。他也没有。整个片场安静了两秒。旁边的场务后来跟化妆师说："那感觉不像演的。"
- 当天晚上他给纪诗然打了视频电话。纪诗然接起来说"今天拍吻戏了？"她看到了片场路透。他说"对。工作需要。"然后两人沉默了好几秒。他知道这个沉默里有她不方便说出口的话。而他不愿意解释。
- 效果：吻戏在技术上不是出轨。但吻完之后两个人的沉默，是所有旁观者都能读懂的暗示。

### 手段7：替身退掉——"我来"
- 某场落水戏，剧组准备了替身。女主说不用。她自己拍。
- 水是零下的。她下水拍了四条。上来的时候嘴唇发紫，全身发抖。江时晏拿着大衣等在岸边。
- 他给她披上大衣时说了一句："你不需要这么拼。"
- 她牙齿还在抖，但笑了——"因为是你来救我的那场戏。我不想别人替我。"这句话什么意思——她可以解释为"想和男主本人对戏"。但她没有解释。她把大衣裹紧，进了保姆车。他在岸边站了很久。
- **与前两线区别**：不是示弱——是展示极端的强度和投入。他看到的不是需要保护的女人，是一个让他敬佩然后心疼的女人。层次不同。

### 手段8：最后一场——让台词就是真心
- 她的杀青戏。剧本里她最后一次见男主。她要说的台词是——"我会走的。但你要记得，这个世界上有一个人，从来没后悔过。"
- 她拍了。导演喊了卡。但她的眼泪没有停。她走出镜头，经过他身边，用只有他能听到的声音把台词又念了一遍——"你要记得，这个世界上有一个人，从来没后悔过。"然后她离开了片场。当天晚上她手机关机。他发不了消息。他喝了酒。
- 效果：用"杀青"的仪式感作为情感引爆点。她不是在拍最后一场戏。她是在把自己的真实情感和角色重叠，让他分不清什么是戏什么是人。这是只有"演员身份"才能实现的——前两线做不到。

## 女主"坏"的展现

**场景一：直播时的眼神**
- 那场300万人观看的直播。她说那句台词时看着他的眼睛——她在镜头前建立了一个"所有观众都看懂了但没人能证明"的暧昧宣言。他女友后来被记者问到回应，她只能说"那是角色台词"。她知道那不是。只能装作不知道。

**场景二：探班日——原配看吻戏**
- 纪诗然来探班。女主那天每条戏都是一条过。纪诗然站在监视器后面，看着这个比自己年轻的女演员用更深的凝视看着她的男朋友。导演拍完说"浅浅你把我的男主拿捏死了"。所有人都在笑。纪诗然也跟着笑。但那个笑容——是她职业生涯里最难的一次表演。

**场景三：模仿原配——他在她身上看到了女友的影子**
- 她研究了纪诗然所有的公开资料。开始在不经意间做纪诗然才做的动作——那个他在家里看了三年的、只有他知道的隐秘习惯。他第一次注意到时以为是巧合。第二次愣住了。第三次他没有说任何话，但他看她的眼神变了。不是看一个新人演员。是在看一个让他困惑的——她怎么变成了她的样子。

**场景四：他替她说过的话，从未替女友说过**
- 江时晏为她发的那条反网暴微博上了热搜。纪诗然打电话问"你为什么要替她说话"。他说"她是被冤枉的"。纪诗然说"那我呢。我被你的粉丝骂了三年，你为我发过一条微博吗"。他沉默了。那个沉默，比任何回答都更让一个女人绝望。

**场景五：分手电话——升级版**
- 江时晏终于打给纪诗然提出分手。电话里纪诗然哭了——"是因为她吗"。他说"不是。是我不爱你了。"——被另一个女人逼到说出口的"不爱了"。分手公布后她打给他——"恭喜你呀，江老师。"语气轻快得像在祝他杀青快乐。他愣了。她已经挂了。

| | 目标1（陆行舟） | 目标2（陈叙川） | 目标3（江时晏） |

| 掩护身份 | "我是来向学长请教的" | "我们是发小/兄弟" | "我在演戏——cut了就收工" |

## 女主"玩狗"手段——得到后的掌控与羞辱

### 核心心态
她对他没有感情。他的全部价值是"被征服的顶级战利品"。征服之后，他的用途从"目标"降级为"工具"——供她取乐、供她消遣、供她在需要时使用。3000万粉丝跪着追的顶流，在她面前随叫随到——这才是她真正享受的东西。

### 手段1：召之即来，挥之即去
- 他刚结束16小时的通告，累到在保姆车上睡着。她的消息弹出来：「睡不着。想见你。」
- 他让司机掉头。开了一个多小时到她的公寓。她开门时穿着睡衣，头发乱着，看起来确实失眠。他陪她躺在床上，她窝在他怀里不到十分钟就睡着了。他在黑暗里看着天花板——自己开了两个小时的车，明天早上六点还有通告。但他没有抽手。怕吵醒她。
- 第二天中午她发了条微信：「昨晚谢谢你呀～」配一个猫猫表情。他回了一句"好好休息"。她在屏幕这头笑了一下——他可能以为她需要他。她只是想知道他会不会来。测试通过。

### 手段2：公开擦边，私下否认
- 狗仔拍到他深夜出入她公寓。照片曝光后上了热搜。经纪人问他"你们什么关系"。他说"普通朋友"。
- 她也被记者堵了一次。记者问她和江时晏的关系。她眨了一下眼睛：「江老师是我很敬重的前辈。希望大家多关注作品。」
- 当天晚上他给她打电话——"新闻的事，没有影响到你吧"。她说"没有呀。不过你经纪人不让你谈恋爱吧。"他沉默了一下。她在沉默里听到了他想说但不能说的话。然后她打了个哈欠：「困了。晚安。江老师。」她把"江老师"三个字拖得很黏。
- 效果：她在公共场合永远叫他"江老师"——最安全也最刺激。因为只有他一个人知道，昨天晚上她叫的是"时晏"。

### 手段3：让他吃醋，让他送上来
- 她和另一个年轻男演员合作新项目。在微博上发了一张两人在片场的合照——站在导演旁边，隔了两个人。但她穿了一件他上次落下的卫衣。
- 江时晏看到了。他打了她的电话。被挂断。他发消息：「那是我的衣服。」她回：「是吗？太久了，我忘了是谁的了。」然后她把手机反扣在桌上，去洗了个澡。出来时屏幕上是他打了十几个未接。最后一条消息是：「别穿别人的。」
- 效果：一句话让他从"想管她"变成"表明态度"。那件衣服后来成了他的标记——每次她穿，狗仔拍到，CP粉都会标记"江时晏的衣服"。她什么都没说。所有证据都是他留下的。

### 手段4：当狗溜——公共场合的隐秘掌控
- 某次电影节。他是红毯压轴。她是新人，座位在第三排。他走过来和她握手，在松开的时候，小指勾了一下她的掌心。这个动作是所有镜头的死角。只有两个人知道。
- 庆功宴上他被人群包围。她走过去敬酒，态度大方："江老师，敬您一杯。谢谢您在剧组对我的照顾。"所有人都觉得这是客套。只有他知道——她说"照顾"的时候，眼里有一闪而过的光。那道光的意思是——等下散场了，我在车里等你。
- 他在凌晨一点结束了应酬。推掉了一群投资人的第二场酒。司机把他放在她公寓楼下。他按门铃的时候，她在监控里看了十秒钟。然后赤脚走过去开门。他第一句话是——"我推掉了所有人的酒。就为了见你。"
- 她笑了一下："进来吧。"
- 效果：他推掉的投资人里，有一个是他下一部电影的合作伙伴。她让他损失了一个未来项目。她不在乎。他也不在乎了。

### 手段5：永远不给他"确定"
- 他问她："我们到底什么关系。"
- 她窝在沙发上翻剧本，头也不抬："你觉得呢。"
- "我想听你说。"
- 她抬起头，看了他几秒。那双眼睛里面有什么一闪而过——他分不清是感动还是玩味。然后她笑了一下："你就是我的——最喜欢的前辈呀。"
- "最喜欢"在这里是一把钝刀——不是没有答案，是答案不够。但他无法追问。因为她是笑着说的。他在她的笑里找不到任何缺口。
- 效果：永远不给他"女朋友"的名分。永远是"前辈""江老师""最喜欢的人"。永远差一步。他就会永远往前追。

### 手段6：让他对别人撒谎——驯服的最终证明
- 某天晚上，他的经纪人打电话来："你最近在浅那边过夜太多。万一被拍——"
- 他说："我们没有在谈恋爱。"
- 经纪人："那为什么你的车停她楼下三天了。"
- 他沉默。林浅浅在旁边听着。她躺在沙发上，把脚搁在他的大腿上，正用他的手机给绵绵看动画片。他转头看着她。她没抬头。然后他对电话说："以后不去了。"
- 挂了电话后他坐到她身边。她说："以后不来了？"他说："骗他的。"
- 她弯了一下嘴角。用只有自己能听到的声音说了两个字：「好狗。」
- 效果：她最享受的不是他爱她。是他为了她对自己人说谎。那是信任破碎的声音。

### 手段7：离开——最后的驯服
- 她消失那天，他正在拍一场重头戏。经纪人不敢告诉他。拍完之后他看到手机上她的消息：「江老师，我要去国外进修一段时间。信号不好。别找我。」
- 他拨了无数次。无法接通。微博已注销。经纪公司说她主动解约了。
- 他翻了她所有的社交媒体。没有留下任何照片。只有CP粉还在不断产出——那些截图和gif是唯一证明她存在过的痕迹。
- 他发了一条微博，配图是一张模糊的片场照——盒饭和一个剧本。剧本的空白处，她用荧光笔画了一颗很小的心。配文只有两个字：「等你。」
- 他没有说在等谁。但CP粉全部懂。
- 他永远不知道——她走之前，故意把那个荧光笔画的爱心放在剧本边缘。她知道他会翻。她知道他会发。她让他自己在3000万粉丝面前，公开表达对一个已经消失的人的等待。那是他自愿的。也是她留给他最后的——训练完毕的狗，不需要主人叫，自己就会跑。

## 女主"惩罚"手段——网暴事件

### 背景
她的身份被扒。极端粉丝扒出了她的住址、她经纪公司的电话、她之前的网剧作品。她的微博评论区被刷屏几十万条——"心机女""蹭热度""滚出娱乐圈""离江时晏远一点"。有人给她经纪公司寄了恐吓信。她的经纪人陈姐被骚扰电话打到关机。三天之内她瘦了一圈，眼眶下面有明显的青——不是演的。是确实没睡好。

但她没有关评论。没有报警。没有发声明。

### 惩罚的核心逻辑
她不恨这些粉丝。她和他们一样清楚——这个世界上最能伤害江时晏的人，不是他的粉丝，不是他的公司，不是纪诗然。是她。她不需要对抗他的粉丝。她只需要惩罚他。粉丝捅的刀，她让他在床上跪着把刀拔出来。

### 手段1：吓他——"我害怕"
- 她连续几天不接他电话。他发了无数条消息，只得到一条回复：「我没事。你别找我了。我不想让你的粉丝再误会。」
- 她用了"误会"这个词。这是最狠的——不指责他，也不指责粉丝。只是说"误会"。等于把所有的伤害压缩成一个无法争辩的模糊地带。
- 他晚上直接开车到了她公寓楼下。她不让他上楼。他在车里坐了一整夜。第二天早上她拉开窗帘看到他还在。车顶落了一层薄雪。
- 她让他上来了。他进来的时候身上是冷的。她递给他一条毯子。然后说了一句话，声音很轻——「我现在一出门就有人追着我骂。我觉得很害怕。我真的——不想牵扯进你的世界了。」

### 手段2：让他内疚——把粉丝的刀扎在他身上
- 她手机上存了几十条截图。网暴最凶的那几条她都存了。她选了几条给他看——不是最恶毒的，而是精准的。"你不配站在江老师旁边""你也不照照镜子""新人别想蹭热度上位"。
- 她给他看的时候哭了。眼泪是真的——她已经几天没睡好，熬也熬出眼泪了。但她选这个时机哭是计算过的。在他最焦虑、最愧疚、最想弥补一切的时刻。
- 他看完之后把她的手机放下。站起来走到窗边。沉默了很久。然后他说：「这些是因为我。」
- 她没有回答。她的沉默等于默认。等于——是的。因为你。

### 手段3：惩罚——你让我被骂，你就要跪
- 那天晚上她没有让他走。也没有让他上床。她让他坐在床边的地板上。
- 她说：「你知道这几天我最生气的是什么吗。」他摇头。她说：「不是被骂。是——你什么都没做。」
- 他被这句话钉住了。她说得对。他什么都没做。他只发了一条不痛不痒的"请理性追星"。那不是在帮她。那是在安抚自己的粉丝。
- 她说：「给我倒杯水。」他站起来去倒了。端回来的时候她看着水杯——「太凉了。」他去重新烧了一壶。端回来。她抿了一口。「有点烫。」他端回去兑凉水，再端回来。这次水温刚好。
- 她喝了一口，把水杯放在床头柜上。看了他一眼：「坐地上吧。地上凉快。」
- 他坐在地板上。背靠着她的床。她在床上玩手机。两小时。他没有起来。最后是她困了，把被子一角从床上垂下来搭在他肩上。说了两个字：「睡吧。」

### 手段4：让他公开站队——粉丝的偶像跪在她脚边
- 第二天他发了一条微博。没有和经纪人商量。没有经过公关团队审核。只有一行字：「我的私人生活不需要别人替我管理。对我的朋友进行人身攻击的人，不是我的粉丝。」
- 炸了。掉的粉以十万为单位。超大面积脱粉。热门话题#江时晏脱粉#。他的经纪人气到砸了茶杯。
- 他发完之后给她打了个电话。「好了吗。」
- 她说：「好了呀。」然后她笑了——「谢谢你。现在——全世界都知道你选我了。」
- 他为了她——亲手砍掉了自己粉丝基础的一部分。这才是真正的惩罚。不是让他难受，是让他付出不可逆的代价。她让他失去了一些永远回不来的东西。而他觉得那是应该的。

### 手段5：奖赏——跪够了才配得到
- 晚上他来找她。她开了门。他站在门口，看起来像一只被暴雨淋过的犬——和自己的粉丝对骂了一天，掉了二十万活粉，经纪人说他疯了。
- 她没说话。踮起脚吻了他。这是她第一次主动吻他。不是角色的吻。不是排练。是林浅浅本人。
- 他愣住了。然后他把她摁在门上，吻回来。更用力。他吻的时候她在想——跪完之后给的奖赏，他会记一辈子。以后每次她需要他付出代价的时候，他都会想到这个吻。

### 第1章：片场

- 游戏环节她说了一句角色台词，但看着他的眼睛说的

- 第一次感情戏排练。两人在排练室独处

### 第4章：私生
- 他的粉丝开始网暴她。评论区几万条骂
- 她回了一条骂她的——"谢谢鞭策"
- 他看到了。他发了一条微博替她挡子弹。纪诗然打电话质问——"你为我发过吗"

### 第5章：探班
- 纪诗然来探班。当天正好有吻戏

### 第6章：综艺
- 两人被安排参加户外真人秀。节目组设计亲密任务

### 第7章：杀青
- 她的杀青戏。剧本台词："你要记得，有一个人从来没后悔过"
- 她从镜头前走下来，经过他身边，用只有他听到的声音把台词又念了一遍
- 手机关机。他喝了一整晚酒

### 第8章：离场
- 纪诗然提了分手。他说"是我不爱你了"
- 她接到他的电话——"我分手了"。她说"恭喜你呀，江老师"
- 第二天她的所有联系方式注销。经纪公司说她决定去国外进修
- 绵绵问"妈妈我们去哪"——"去下一个片场"
- 系统评定：SSS级
